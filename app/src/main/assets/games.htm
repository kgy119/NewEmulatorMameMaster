Supported Games

MAME4droid (Current) uses game ROMs from the current desktop MAME. Note: This is different than the ROMs used in MAME4droid (0.139u1). These ROMs cannot be used interchangeably.

ROM Location and Naming Convention

ROMs should be placed in the "roms" folder of your MAME4droid installation folder. In a default install of MAME4droid, this path is /storage/emulated/0/Android/data/ com.ing.webhard/files/roms. You should copy your roms from a PC using the Android Debug Bridge (ADB) tool

Alternatively, you can select the external storage option at the MAME4droid installation time and indicate a directory where your roms are or are going to be stored. The roms will be slower to read but they will not be deleted when you uninstall the application. You can also copy them with a android file manager.

Additionally, you can copy other selected resources such artworks/samples into directories under the external storage ROMs folder so that they are also read from the external storage. Also, if you are an advanced user, you can modify the .ini files on the main installation folder and configure everything on external storage

Anyway, ROMs may be stored anywhere you like in a later time, but you'll need to go to Options->Settings->General->Change ROMs path

Note: File and directory names are case-sensitive. MRDO.ZIP is not the same as mrdo.zip. Always use ZIPped ROMs with filenames in lower case..
